# linux高性能网络编程

* 使用**线程池 + 非阻塞IO + epoll边沿触发IO复用 + Reactor事件处理模式 + 半同步/半反应堆模式**的并发模型
* 实现了一个固定线程数的半同步/半反应堆并发模式的线程池
* 使用一个工作队列,解除了主线程与工作线程的耦合,工作线程之间通过竞争来取得任务并执行
